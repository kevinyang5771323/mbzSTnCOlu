# 项目简介

## 核心功能
- 数据采集
- 数据预处理
  - 数据清洗
  - 处理缺失值
  - 格式化数据
- 数据分析
  - 使用 Pandas 和 NumPy 进行数据处理和统计分析
  - 多维度数据分析
- 数据可视化
  - 使用 Matplotlib、Seaborn 或 ECharts生成图表
  - 将图表嵌入到 Web 页面中
  - 使用 pyecharts 库生成图表对象
  - 将生成的图表嵌入到 Flask 的 HTML 模板中
- 系统管理模块
  - 管理员功能
    - 用户管理
    - 网站信息管理
    - 权限控制
  - 后台管理系统
    - 用户管理
    - 数据管理
    - 权限管理

## 适用场景
- 电影数据分析
- 电影网站内容管理
- 用户信息管理

## 技术特点
- 前端：使用 Flask 框架和 HTML/CSS/JavaScript 开发
- 后端：使用 Python 脚本进行数据处理和爬取，使用 MongoDB/MySQL 存储数据
- 数据分析：使用 Pandas 和 NumPy 进行数据分析和统计
- 数据可视化：使用 Matplotlib、Seaborn 或 ECharts 以及 pyecharts 库进行图表生成和展示
- 系统管理：使用 Django 框架实现后台管理系统
